## 1. 비회원 모드 슈퍼관리자 등록 시 "삭제됨" 표시 수정

배경: 비회원 모드에서 슈퍼관리자가 스케줄 등록 시 `user_id`에 슈퍼관리자 ID가 저장됨.
슈퍼관리자는 `tenant_members`에 없으므로 `withdrawnUserIds`에 포함되어 "삭제됨" 표시.
수정: 비회원 모드에서는 `user_id = null`로 저장하도록 변경.

- [ ] 비회원 모드(직접입력) 조직에 슈퍼관리자로 접속
- [ ] 스케줄 셀 클릭 → 이름 직접 입력 후 등록
- [ ] 등록된 배정에 "삭제됨" 표시가 나타나지 않음
- [ ] **[DB 확인]** `SELECT user_id FROM assignments ORDER BY created_at DESC LIMIT 5;` — 방금 등록된 행의 `user_id`가 `null`임

---

## 2. 회귀 테스트

- [ ] 회원공유 모드에서 관리자가 직접 등록 시 `user_id = admin_id`로 정상 저장 (기존 동작 유지)
- [ ] 회원공유 모드에서 회원 선택 후 등록 시 `user_id = 선택된_회원_id`로 정상 저장
- [ ] 비회원 모드에서 기존에 등록된 배정(user_id = null)은 영향 없음
- [ ] 삭제됨 표시 로직: 실제로 탈퇴/삭제된 회원 배정에는 여전히 "삭제됨" 표시
