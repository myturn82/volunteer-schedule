## 1. 주달력 칩 배경색 월달력 통일

배경: 월달력 `plus` 열은 50plus 배정에 `tint-plus`(살구색) 배경을 사용하지만,
주달력은 배정 타입 무관하게 `tint-sun`(노란색)만 사용해 색상이 달랐음.
수정: WeekGrid 비분할 모드에서 `50plus` 타입 배정은 `tint-plus` 색상으로 표시.

### 50plus 배정 색상

- [ ] 50plus 타입 배정을 월달력에서 등록 → plus 열에 살구색(`tint-plus`) 칩으로 표시됨
- [ ] 같은 배정을 주달력에서 확인 → 동일하게 살구색 칩으로 표시됨 (기존에는 노란색)
- [ ] 50plus 배정만 있는 셀 → 셀 전체 배경도 살구색
- [ ] 일반 회원(member) 배정은 기존과 동일하게 노란색(`tint-sun`) 유지

### 혼합 셀 (member + 50plus)

- [ ] 한 셀에 일반 회원 + 50plus 배정이 함께 있을 때:
  - 셀 전체 배경: 노란색(sun tint, 일반 회원 기준)
  - 일반 회원 칩: 노란색
  - 50plus 칩: 살구색
- [ ] 하이라이트(이름 검색) 시 양쪽 모두 노란 형광 배경으로 정상 표시

### 야간 시간대 (20시 이후)

- [ ] 야간 슬롯 일반 회원 배정: 월/주 모두 보라색(`tint-moon`)
- [ ] 야간 슬롯 50plus 배정: 주달력에서도 살구색(`tint-plus`)

## 3. 회귀 테스트

- [ ] 일반 회원(member) 배정의 월/주달력 칩 색상 변화 없음
- [ ] 역할 분할(split role) 모드에서 주달력 칩 색상 기존과 동일 (`ROLE_TINTS` 순환)
- [ ] 고정된 배정(`is_locked`) 잠금 아이콘 정상 표시
- [ ] 탈퇴/삭제 회원 배정 "삭제됨" 표시 정상

## 테스트 우선순위

1. **1번 (50plus 색상)** — 핵심 수정 사항
2. **1번 (혼합 셀)** — 경계 케이스
3. **2번 (비회원 회귀)** — 이전 버그픽스 영향 없음 확인
4. **3번 (회귀)** — 기존 색상 변화 없음 확인
